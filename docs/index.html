If you see 404, turn Pages on and pick one method

A) GitHub Actions (matches this repo’s workflow):
Repository Settings → Pages → Build and deployment → Source: GitHub Actions → select workflow “Publish GitHub Pages preview” (if you see a dropdown) → wait until the latest run in Actions is green (1–2 min), then try the URL again.

B) No Actions (simpler): same Settings → Pages → Source: Deploy from a branch → Branch: main → Folder: /docs → Save. Do not use (A) and (B) at the same time.
