Turn on the site (branch + /docs, no Action needed)

  1. Open Settings → Pages.
  2. Under Build and deployment → Source choose Deploy from a branch (not “GitHub Actions”).
  3. Branch = main, folder = /docs → Save. Wait 1–3 minutes, then open your site in a new tab.

Your project URL is: https://nellcodx.github.io/reservation/ — the path is your username and repo name (all lower case). If you were using GitHub Actions as the Pages source before, switch it to Deploy from a branch or the old deploy never went live and you will keep seeing 404.
